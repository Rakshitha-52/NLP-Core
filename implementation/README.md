# NLP Implementations

This directory contains hands-on implementations of Natural Language Processing (NLP) concepts using Python and machine learning.

Each project in this folder focuses on applying theoretical concepts to real-world problems, with an emphasis on clean code, experimentation, and evaluation.

## Projects

### 1. Spam Guard (Spam Detection System)
A machine learning-based spam detection system that classifies messages as spam or not spam.

- Uses CountVectorizer for text feature extraction
- Implements Multinomial Naive Bayes for classification
- Evaluates model performance using classification metrics
- Supports custom input predictions

Location: `spam_guard/`

---

## Tech Stack
- Python
- Scikit-learn
- Pandas
- NumPy

---

## Goal
The goal of these implementations is to strengthen practical understanding of NLP techniques and build real-world problem-solving skills.

Each project is designed to be simple, clear, and extendable for further experimentation.
