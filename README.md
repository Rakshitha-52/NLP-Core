# NLP-Core
Collection of NLP concepts and practical implementations including text classification and machine learning-based projects.
This repository contains my learning and hands-on implementations in Natural Language Processing (NLP).

It is structured to cover both foundational concepts and practical applications:
- concepts → notebooks and notes covering core NLP techniques
- implementations → real-world projects applying NLP with machine learning

The goal of this repository is to build a strong understanding of text processing, feature extraction, and model development through practical experimentation.
